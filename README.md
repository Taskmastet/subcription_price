# Цѣна Жизни · сколько часов жизни стоят подписки

Мрачный готический калькулятор: считает, сколько **часов твоей жизни** уходит
на подписки, исходя из твоей почасовой ставки.

## Логика

1. Вписываешь стоимость подписок (в месяц) и свой заработок за час.
2. Сайт считает:
   - `часов в месяц`  = сумма подписок ÷ ставка за час
   - `часов в год`    = ×12
   - `рабочих дней (8ч) в год` = часов в год ÷ 8
   - `полных суток за 10 лет`  = часов в год × 10 ÷ 24
3. Выносит «приговор» — драматичную строку в зависимости от масштаба.

Валюта переключается (₽ / $ / € / ₴); поскольку цена и ставка в одной валюте,
на результат-в-часах она не влияет — только на подпись.

## Визуальный референс

Классическая готика: чёрный гранжевый фон, кроваво-красный, готический
blackletter с потёками, золото-бронзовые орнаменты и витые рамки, ритуальный
круг-сигил по центру. Вдохновение — Alchemy Gothic, *Alone in the Blood*,
Rapier Tristesse, атмосфера A24.

Шрифты (Google Fonts): Pirata One, Cinzel Decorative, Cinzel, EB Garamond.

## Запуск

Чистая статика — без сборки. Открой `index.html` в браузере
(или подними локальный сервер: `python3 -m http.server`).

## Файлы

- `index.html` — разметка
- `styles.css` — готическая тема
- `script.js`  — логика расчёта
